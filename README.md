# Manna

Manna is a local-first, extensible personal AI agent platform.

At a high level, it can reason over multi-step tasks, call specialized tools, and orchestrate workflows across domains such as coding, research, document processing, data access, and knowledge graphs.

## Canonical documentation

The **only source of truth** for setup, architecture, API behavior, tools, environment variables, and operational details is the VitePress documentation in [`/docs`](/docs).

Start here:
